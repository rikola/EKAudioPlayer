EKAudioPlayer

Framework for playing music from the iTunes library.

View on GitHub Download .zip Download .tar.gz

An easier way to play music from the iTunes library.

This framework includes an easier way to access the music library. It has lots of classes to help handle the logic associated with sorting MPMediaQueries and integrating the views into iOS apps.
